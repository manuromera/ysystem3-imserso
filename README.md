# Paquete para CODEX — Módulo IMSERSO to the Limit para YSYSTEM3

Este paquete está pensado para que CODEX implemente un **módulo Foundry** (no un sistema separado) que transforme `ysystem3-srd` en una experiencia de IMSERSO to the Limit.

## Contenido
- `data/module.json.template` — plantilla base del manifiesto del módulo
- `data/imserso-theme-tokens.json` — paleta, motivos y targets de CSS
- `data/imserso-localization.json` — labels, textos y ayuda contextual
- `data/imserso-ui-map.json` — mapa de sustituciones visuales/terminológicas
- `data/imserso-generator-tables.json` — tablas para generador de PJ/PNJ
- `assets/branding/*` — branding y fondos originales listos para usar
- `assets/icons/*` — iconos originales de tipos de ítem
- `docs/ARQUITECTURA_MODULO_IMSERSO.md` — estrategia técnica
- `codex/CODEX_PROMPT_MODULO_IMSERSO.md` — prompt directo para CODEX

## Nota sobre el tono visual
Los assets incluidos NO copian portadas comerciales. Son recursos originales inspirados en:
- viajes del IMSERSO
- postal mediterránea
- costa, hotel y autocar
- bingo, pastilleros y cachivaches de abuela/abuelo
